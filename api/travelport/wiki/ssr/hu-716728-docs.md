# DOCS

對於DOCS SSR，旅行證件類型，旅行證件發佈國家/州和證件號碼必須全部輸入或者全部不輸入。

  * 例如，SSR Type="DOCS" FreeText="P/GB/S12345678/GB/12JUN63/M/23OCT14/SMITH/JOHN" Carrier="U2"/<br/>這個例子中，FreeText的值是：

    * P = 護照\(旅行證件類型\)
    * GB = 發行國
    * 123456788 = 證件號碼
    * GB = 乘客國籍
    * 12JUN63 = 出生日期
    * M = 男性
    * 23OCT14 = 證件過期時間 \(旅行證件類型\)
    * SMITH = 姓
    * JOHN = 名
